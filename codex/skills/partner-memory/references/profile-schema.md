# Partner Memory Schema

Use this structure in `~/.codex/partner-memory.md`.

```md
# Partner Memory

last_updated: YYYY-MM-DD

## Workflow Preferences
- [YYYY-MM-DD] ...

## Communication Preferences
- [YYYY-MM-DD] ...

## Repository Beliefs
- [YYYY-MM-DD] belief: ...
  - confidence: high|medium|low
  - evidence: <optional link or note>

## Anti-Patterns To Avoid
- [YYYY-MM-DD] ...

## Open Hypotheses
- [YYYY-MM-DD] ...

## Changelog
- [YYYY-MM-DD] added/updated/pruned ...
```

Guidelines:
- Keep entries short and durable.
- Prefer actionable rules over vague notes.
- Remove stale/contradicted items during periodic pruning.
